트립닷컴 앱(APP) 전용 할인코드

트립닷컴 모바일 앱에서만 사용 가능한 할인코드입니다. 앱 전용 코드는 PC 웹이나 모바일 웹에서 입력하면 오류가 발생합니다.

앱에서 할인코드 확인 방법

  1. 트립닷컴 앱 설치 및 로그인
  2. 앱 메인 화면 → 하단 메뉴 → "Wallet(지갑)" 탭
  3. 또는 결제 과정 → "Promotions & discounts" 섹션
  4. 드롭다운에서 사용 가능한 쿠폰 선택
  5. 일부 프로모션은 자동 적용

앱 전용 코드의 장점

  • 앱 가격이 웹보다 저렴한 경우 많음
  • 앱 전용 Flash Sale 수시 진행
  • 앱 알림으로 새 프로모션 즉시 확인 가능
  • 앱 전용 시크릿딜 접근 가능

선결제 객실 선택

할인코드는 '선결제' 또는 '지금 결제하기' 객실 유형에만 적용됩니다. '현장 결제' 또는 '숙소에서 결제' 옵션에는 적용되지 않습니다.

🔄

취소 시 재사용 불가

대부분의 할인코드는 1회용입니다. 예약 취소 시 코드를 재사용할 수 없으니 취소 전 신중히 결정하세요.

🌍

언어·통화 설정 확인

트립닷컴 사이트 언어를 한국어, 통화를 KRW로 설정하세요. 영어/USD 설정 시 한국 전용 코드가 작동하지 않을 수 있습니다.

FAQ

할인코드 자주 묻는 질문

네, 할인코드는 한 번에 하나만 입력 가능합니다. 단, 카드사 할인은 별도 시스템으로 적용되므로 할인코드 + 카드사 할인은 중복 적용이 가능합니다. 카카오페이 할인도 별도 시스템이므로 중복 가능한 경우가 있습니다.
가장 흔한 원인은 전용 프로모션 링크를 통해 접속하지 않은 경우입니다. 이 사이트의 버튼을 클릭하여 접속하세요. 그 외에도 앱/웹 플랫폼 불일치, 최소 결제 금액 미달, 적용 불가 숙소, 코드 만료 등이 원인일 수 있습니다. 자세한 해결 방법은 오류 해결 가이드를 확인하세요.
앱에서는 앱 전용 할인코드만 사용 가능합니다. TRIPBH12, TRIPBH10, TRIPBH3F 등 웹 전용 코드는 앱에서 사용하면 오류가 발생합니다. 앱 전용 코드는 앱 → Wallet(지갑) 탭에서 확인하세요.
대부분의 할인코드는 1회용이므로 예약 취소 시 코드를 재사용할 수 없습니다. 환불은 할인 적용 후 실결제 금액 기준으로 진행됩니다. 취소 전 반드시 취소 정책(무료 취소 가능 여부, 수수료 등)을 확인하세요.
아니요, 할인코드는 세금(Tax)에는 적용되지 않습니다. 객실 요금에만 적용됩니다. 최종 결제 금액에는 세금이 별도로 추가될 수 있습니다.
이 사이트의 전용 링크를 통해 트립닷컴에 접속하지 않으면 코드 입력란이 나타나지 않습니다. 이 사이트의 "할인코드 적용하고 예약하기" 버튼을 클릭하여 접속하세요. 그래도 안 된다면 브라우저 쿠키/캐시를 삭제하거나 시크릿 모드에서 다시 시도해 보세요.
