## 1. 트립닷컴 소개 및 할인코드 개요

### 1.1 트립닷컴이란?

- 2003년 나스닥 상장, 중국 Ctrip 계열의 글로벌 온라인 여행사 (OTA)
- 전 세계 4억 명 이상의 회원 보유
- 140만 개 이상의 숙소, 200만 개 이상의 항공편 제공
- 항공권, 호텔, 기차표, 렌터카, 액티비티, 공항픽업 등 종합 여행 서비스
- 연중무휴 24시간 다국어(한국어 포함) 고객센터 운영
- 실시간 PC/앱 채팅 서비스 지원

### 1.2 할인코드 개요

트립닷컴 할인코드란 예약 결제 단계에서 입력하면 즉시 할인이 적용되는 프로모션 코드를 말한다. 할인코드는 크게 4가지 유형으로 나뉘며, 각각 적용 조건과 채널이 다르다.

**핵심 특징:**
- 체인호텔(하얏트, 메리어트, 어코르, 힐튼 등)에도 할인코드 적용 가능 → 타 OTA 대비 최대 강점
- 대부분 전용 프로모션 링크를 통해 접속해야만 적용 가능
- 하나의 예약에 두 개 이상의 할인코드 중복 사용 불가
- 선착순 소진 방식이 많아 조기 마감될 수 있음

## 3. 상품별 할인코드 상세

### 3.1 항공권 할인코드

**일반 항공권 할인코드:**
- 트립닷컴에서 항공권 예약 시 결제 직전에 코드 입력
- 할인은 세금·발권 수수료 등 기타 요금 제외 후 **순수 항공운임**에서 적용
- 인플루언서 코드 기준 약 3% 할인, 한도 없이 사용 가능

**신규회원 항공권 할인:**
- 첫 예약 시 최대 21,000원 할인
- 21만원 이상 예약 → 5,000원 할인
- 52만원 이상 예약 → 21,000원 할인

**항공사 전용 특가:**
- 이스타항공 전용 1만원 할인코드
- 이스타항공 전용 전 지역 10% 호텔+항공권 할인코드
- 티웨이항공 수요일 특가 세일
- 제주항공, 아시아나항공, 대한항공, 카타르항공 등 항공사별 특가

**항공권 가격 동결(Price Lock):**
- 소액 보증금 지불 시 항공권 요금을 14일간 동결
- 가격 상승 시 트립닷컴이 차액 보전
- 가격이 내려가면 자동으로 더 낮은 금액 적용
- 성수기 인기 노선 예약 시 특히 유용

### 3.2 호텔 할인코드

**일반 호텔 할인코드:**
- 인플루언서 코드: 최대 12% 할인
- 전용 링크 경유 필수, KRW 결제 가능
- 체인호텔(메리어트, 힐튼, 하얏트, 어코르 등)에도 적용 가능

**신규회원 호텔 할인:**
- 첫 호텔 예약 시 최대 10% 할인
- 추천코드 입력 시 호텔 예약 8% 할인 쿠폰 추가 제공
- 45만원 이상 결제 시 최대 3만원 할인
- 3연박 이상 예약 시 5% 할인 (최대 18,500원)

**회원등급별 호텔 할인:**
- 비로그인 → 로그인만으로도 상당한 가격 차이 발생
- 실버: 최대 5% 할인
- 골드: 최대 10% 할인
- 플래티넘: 최대 15% 할인
- 다이아몬드: 최대 20% 할인

### 3.3 액티비티(투어·티켓) 할인코드

- 신규 회원 5% 할인 (최대 4,000원)

- 매주 금요일 1+1 프로모션 업데이트
- 롯데월드, 상하이 디즈니랜드 등 인기 테마파크 1+1 특가 수시 진행

### 3.4 렌터카 할인코드

- 트립닷컴에서 렌터카 첫 예약 시 최대 8% 할인
- 결제 전 할인코드 입력으로 즉시 적용
- 앱 전용 발급인 경우 있음

### 3.5 공항픽업/샌딩 할인코드

### 7.1 웹사이트(PC) 적용 방법

1. **프로모션 전용 링크로 접속** (반드시 전용 링크 경유)
2. 원하는 상품(호텔/항공권 등) 선택 후 **"예약"** 클릭
3. 로그인 후 주문 페이지에서 개인정보 입력
4. 예약(결제) 페이지 하단 → **'예약 혜택'** 탭에서 **'할인코드 선택/입력'** 클릭
5. 할인코드 입력란 활성화 → 복사한 코드 붙여넣기
6. **"적용"** 버튼 클릭 → 할인 금액 즉시 표시
7. 할인 적용 확인 후 **"결제"** 버튼 클릭 → 결제수단 선택 → 결제 완료

### 7.2 모바일 앱 적용 방법

1. 프로모션 전용 링크로 앱 접속
2. 상품 선택 → 예약 진행
3. 결제 화면에서 '할인코드' 입력란 확인
4. 코드 입력 → 적용 → 결제 완료

### 7.3 할인코드 확인 방법

- **내 정보 → 할인코드 페이지:** 발급받은 할인코드 목록 확인 가능
- 쿠폰 '발급' 버튼 클릭 → '사용' 버튼 활성화
- My 계정 관리 → 할인코드 탭에서 확인

### 7.4 중요 주의사항

- **반드시 전용 프로모션 링크로 접속해야 함** → 그냥 트립닷컴 홈에서 코드만 입력하면 적용 안 됨
- 한번 적용한 코드는 계속 사용 가능 (코드에 따라 다름)
- 앱 전용 코드는 PC 웹에서 입력창이 보이지 않을 수 있음
- 언어/통화 설정에 따라 표시되는 프로모션이 달라질 수 있음

## 8. 할인코드 적용 오류 및 해결 방법

### 8.1 오류 원인 TOP 5

| 순위 | 원인 | 상세 설명 |
|------|------|-----------|
| 1 | **전용 링크 미경유** | 프로모션 전용 링크로 접속하지 않고 직접 사이트 방문 시 코드 미적용. 포털 검색 결과의 '광고'/'스폰서' 링크 경유 시에도 미적용 |
| 2 | **유효기간 만료** | 예약 기간(프로모션 기간) 또는 숙박/출발 기간 조건 초과 |
| 3 | **상품 불일치** | 호텔 전용 코드를 항공권에 사용하거나, 결합상품(항공+호텔 패키지)에 코드 적용 시도 → 단독 예약에만 사용 가능 |
| 4 | **코드 입력 오류** | 오타, 대소문자 구분, 공백 포함 등 |
| 5 | **최소 결제금액 미충족** | 최소 결제금액·최소 숙박일수·특정 결제수단 조건 등 미충족 |

### 8.2 추가 오류 사례

- **트립코인과 할인코드 동시 적용 불가:** 결제 단계에서 둘 중 하나만 선택 가능. 트립코인 사용 시 할인코드 입력 제한됨
- **채널 인증 실패:** 카드사 제휴 코드의 경우 카드사 이벤트 페이지 경유 필수. "프로모션 채널 인증에 실패했습니다" 오류 발생
- **IE 브라우저 사용:** Internet Explorer에서는 할인코드 입력이 제대로 안 됨 → 크롬, 엣지 등 최신 브라우저 사용 필수
- **캐시 문제:** 브라우저 캐시로 인해 코드 인식 안 되는 경우 있음
- **애드블로커:** 광고 차단 프로그램이 활성화된 경우 할인코드 기능 방해 가능
- **다른 쿠폰/프로모션과 중복 불가:** 이미 다른 프로모션 할인이 적용된 객실에는 추가 코드 적용 불가능한 경우 있음

### 8.3 오류 해결 체크리스트

1. **전용 프로모션 링크로 재접속** (가장 중요!)
2. 할인코드 유효기간 및 사용 조건(예약기간/숙박기간) 확인
3. 적용 대상 상품(호텔/항공/액티비티) 일치 여부 확인
4. 결제수단 조건(카드사, KRW/USD) 확인
5. 최소 결제금액 / 최소 숙박일수 충족 여부 확인
6. 트립코인 사용 해제 후 할인코드 재입력
7. 브라우저 캐시 삭제 후 재시도
8. 애드블로커 비활성화
9. 크롬/엣지 등 최신 브라우저 사용
10. 시크릿(프라이빗) 모드로 재시도
11. 앱 전용 코드는 반드시 모바일 앱에서 입력

## 11. 결제 수수료 절약 방법 (DCC 회피)

### 11.1 트립닷컴 결제 구조 이해

- 트립닷컴은 글로벌 여행 플랫폼 → 기본적으로 **해외 가맹점** 분류
- 아무 설정 없이 결제하면 카드사 해외 이용 수수료 + 이중 환전 수수료 → **최대 8% 추가 비용** 발생 가능
- DCC(Dynamic Currency Conversion) = 자국 통화 결제 서비스: 원화→달러→원화 이중 환전으로 약 3~8% 추가 수수료

### 11.2 수수료 절약 방법

| 방법 | 효과 |
|------|------|
| **통화를 KRW로 설정** 후 결제 | 국내 결제 처리 → 해외 수수료 없음 |
| **간편결제(카카오페이, 네이버페이) 사용** | KRW 국내 결제 처리 |
| **KCP, KG이니시스** 등 국내 PG 통해 결제 | 수수료 없는 국내 결제 |
| **USD로 설정** 후 결제 | 이중 환전 방지 (단, 해외 수수료는 발생) |

### 11.3 통화 설정 변경 방법

**웹:**
- 트립닷컴 홈페이지 우측 상단 통화 버튼(KRW) 클릭 → 원하는 통화 선택

**앱:**
- 마이페이지 → 우측 상단 톱니바퀴(⚙️) 아이콘 → 통화 설정 변경

**최종 확인:** 결제창에 '국내 결제', 'KCP', 'KG이니시스' 등 문구가 보이면 수수료 없는 국내 결제 확정

## 12. 할인 극대화 전략 및 꿀팁

### 12.1 할인 스택 전략

할인코드 단독으로도 유용하지만, 여러 전략을 조합하면 체감 할인율 20~30%까지 가능:

1. **회원등급 특가** (로그인 후 확인되는 회원 전용 가격)
2. **할인코드 적용** (인플루언서 or 카드사 제휴 코드)
3. **간편결제 프로모션** (카카오페이 12% 등)
4. **네이버페이 포인트 0.5% 적립**
5. **수수료 절약** (KRW 설정으로 해외 수수료 회피)

### 12.2 체인호텔 할인 극대화

- 트립닷컴은 메리어트, 힐튼, 하얏트, 어코르 등 체인호텔에도 할인코드 적용 가능
- 타 OTA(아고다, 호텔스닷컴 등)에서는 체인호텔 쿠폰 미적용 → 트립닷컴의 최대 강점
- 고가 호텔일수록 %할인의 절대금액이 커지므로 효과 극대화
- 단, 100% 모든 체인호텔 적용은 아님 → 예약 시 확인 필요

### 12.3 친구 추천 프로그램

- 친구에게 추천 링크 전송 → 가입 및 예약 완료 시
- 본인: 2만원 상당 리워드
- 친구: 12,800원 할인
- 최대 10명까지 초대 가능
- 여러 예약 건 시 친구와 나눠서 예약하면 최소 3만원 이상 절약

### 12.4 항공권 + 호텔 동시 예약 추가 할인

- 항공권 예약 후 호텔 예약 시 '항공권 예약 전용 가격' 적용
- 여기에 호텔 할인코드까지 중첩 가능 → 이중 할인 효과

### 12.5 카카오톡 채널 추가

- 트립닷컴 카카오톡 채널 추가 시 할인 코드 발송
- 호텔/항공권 예약에 사용 가능

### 12.6 멤버스데이

- 트립닷컴 멤버스데이 등 특별 이벤트 시 추가 프로모션 제공

## 14. 자주 묻는 질문 (FAQ)

### Q1. 할인코드는 어디서 확인하나요?
트립닷컴 메인 페이지, 앱 이벤트 영역, 제휴 카드사 프로모션 페이지, 또는 인플루언서/블로그의 전용 링크를 통해 확인할 수 있습니다.

### Q2. 할인코드 중복 사용이 가능한가요?
일반적으로 1회 1건에 1개의 할인코드만 사용 가능합니다. 다른 코드와 중복 적용은 불가합니다.

### Q3. 트립코인과 할인코드를 동시에 사용할 수 있나요?
기본적으로 동시 적용이 불가합니다. 결제 단계에서 둘 중 하나만 선택해야 합니다. (일부 예외 가능성 있으므로 결제 시 확인)

### Q4. 취소 시 할인코드는 어떻게 되나요?
예약 취소 시 사용한 할인코드는 트립닷컴 계정으로 재발행됩니다. 프로모션 기간 내에 재사용 가능합니다. 단, 할인코드로 할인받은 금액 자체는 환불되지 않습니다.

### Q5. 예약 변경 시 할인코드는 유지되나요?
예약을 변경하면 사용한 할인코드는 더 이상 유효하지 않습니다. 할인코드를 다시 사용하려면 예약을 취소하고 다시 예약해야 합니다.

### Q6. 비회원도 예약이 가능한가요?
비회원 예약이 가능한 경우도 있지만, 예약 관리가 번거롭고 회원 혜택(적립/할인/등급)을 받을 수 없으므로 회원 가입 후 이용을 권장합니다.

### Q7. 해외 결제 수수료를 피하려면?
KRW 통화 설정 후 간편결제(카카오페이, 네이버페이)를 이용하면 국내 결제 처리되어 해외 수수료가 발생하지 않습니다.

### Q8. '예약 기간'과 '숙박 기간'의 차이는?
예약 기간(프로모션 기간)은 할인코드를 입력해 실제 예약할 수 있는 기간이고, 숙박 기간은 예약한 호텔에 실제로 체크인하는 날짜 범위입니다. 두 조건 모두 충족해야 코드 적용 가능합니다.

### Q9. 항공권 할인코드의 '출발 기간'이란?
항공권 할인코드에 명시된 출발 기간 내에 출발하는 항공편에만 할인이 적용됩니다.

### Q10. 회원등급은 어떻게 빠르게 올릴 수 있나요?
서울→수원 같은 저렴한 국내 기차표(약 2,700원)를 예약하면 1건의 적격 예약으로 인정되어 즉시 골드 등급으로 승급 가능합니다. 총 4건이면 플래티넘까지 도달합니다.

### Q11. 체인호텔에도 할인코드가 적용되나요?
트립닷컴은 타 OTA와 달리 메리어트, 힐튼, 하얏트, 어코르 등 주요 체인호텔에도 할인코드 적용이 가능합니다. 단, 모든 체인호텔에 100% 적용되는 것은 아니므로 예약 시 확인이 필요합니다.

### Q12. 할인코드 입력란이 보이지 않아요
프로모션 전용 링크로 접속했는지 확인하세요. 앱 전용 코드는 PC 웹에서 입력창이 표시되지 않을 수 있습니다. 또한 '예약 혜택' 섹션 자체가 보이지 않으면 프로모션 URL 접속 여부를 재확인하세요.
